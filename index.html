排帖助手 - 服务条款

最后更新：2026年5月30日

1. 服务说明

排帖助手（以下简称"本应用"）是一个个人工具，使用 TikTok 官方 Content Posting API 帮助用户管理自己的 TikTok 账号内容发布。

2. 使用限制

本应用仅供应用所有者个人使用，不向第三方用户开放。所有通过本应用发布的内容均由用户自行负责，需遵守 TikTok 社区准则和服务条款。

3. 数据隐私

本应用仅在本地存储 TikTok API 授权凭证（Token），用于调用官方 API 发布视频。本应用不会收集、存储或传输任何用户个人信息至第三方服务器。

4. 免责声明

本应用"按现状"提供，不提供任何明示或暗示的保证。用户使用本应用过程中因 TikTok 平台政策变更、API 限制等因素导致的功能异常，本应用不承担责任。
