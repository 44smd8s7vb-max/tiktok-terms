排帖助手 - 隐私政策

最后更新：2026年5月30日

1. 信息收集

本应用不收集任何用户个人信息。OAuth 授权 Token 仅存储在用户本地计算机的配置文件中，用于调用 TikTok 官方 API。

2. 信息使用

授权 Token 仅用于以下目的：

  • 通过 TikTok Content Posting API 上传视频
  • 在用户指定的时间发布视频至用户自己的 TikTok 账号

3. 信息存储

所有数据（Token、排期记录）仅存储在用户本地设备。本应用不连接任何第三方服务器，不上传任何数据至云端。

4. 信息共享

本应用不与任何第三方共享任何信息。所有 API 通信仅发生在用户设备与 TikTok 官方服务器之间。

5. 用户权利

用户可随时删除本地配置文件（config.json）以撤销所有授权，也可在 TikTok 应用内撤销本应用的应用授权。

6. 联系方式

如有隐私相关问题，请联系：admin@example.com
